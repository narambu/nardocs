# Supported Platforms

## UX (Desktop App)

| Platform | Status |
|:---------|:-------|
| macOS (darwin) | Supported |
| Windows | Coming soon |
| VS Code Extension | Coming soon |

Install the DMG from [nar.narambu.com/nar](https://nar.narambu.com/nar). The desktop app includes a built-in editor, terminal, and the full [UX](/nar/ux) control panel.

## CLI

| Platform | Status |
|:---------|:-------|
| macOS (darwin) | Supported — `darwin-arm64` |
| Linux | Coming soon |

Download narcli from [GitHub Releases](https://github.com/narambu/nardownloads/releases). See the [CLI guide](/nar/cli) for setup and usage.

## Architecture

Both the UX and CLI share the same underlying engine. Every action available in the UX can also be run via the CLI — they are functionally equivalent.
