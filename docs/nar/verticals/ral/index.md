# React Auth Lambda

A React + Vite starter with Google OAuth authentication, deployed as a Lambda service. This vertical comes bundled with Nar App — no separate download needed.

**[Getting Started](/nar/verticals/ral/getting-started)** — Configure auth and deploy.

**Includes:**
- **[React UI](/nar/verticals/ral/ui)** — Frontend built with React 19, Vite, and Tailwind CSS.
- **[Auth Service](/nar/verticals/common/auth)** — Python Lambda that handles Google OAuth login and session management.
